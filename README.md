# Heart Disease Prediction

## Objective
Build a machine learning model to predict heart disease using medical and health-related data.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Model
- Logistic Regression

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training and prediction
- Confusion Matrix
- ROC Curve
- Feature Importance Analysis

## Evaluation Metrics
- Accuracy Score
- Classification Report
- ROC-AUC Score

## Result
Successfully developed a heart disease prediction system using Logistic Regression with data preprocessing, visualization, and model evaluation techniques.
